## 5.1.3
Dados dos números: a y b. Calcule la potencia $a^b$, usando sólo
multiplicaciones sucesivas.

$$
mult(a, b) =
\begin{cases}
  1, & \text{si } b = 0, \\
  a, & \text{si } b = 1, \\
  a * (b - 1), & \text{si } b > 1
\end{cases}
$$

```
FUNCION mult(a, b: entero): entero ES
  SI (b = 0) ENTONCES
    mult := 1
  CONTRARIO
    SI (b = 1) ENTONCES
      mult := a
    CONTRARIO
      mult := a * mult(a, b - 1)
    FIN_SI
  FIN_SI
FIN_FUNCION
```
