## 5.1.2
Dado un número n como parámetro de entrada, calcular el n-ésimo número de la
serie de Fibonacci. Tener en cuenta la siguiente definición:

$$
Fib(n) =
\begin{cases}
  1, & \text{si } n = 1, n = 2 \\
  Fib(n - 1) + Fib(n - 2), & \text{si } n > 2
\end{cases}
$$

```
FUNCION fibonacci(n: entero): entero ES
  SI (n = 1) O (n = 2) ENTONCES
    fibonacci := 1
  CONTRARIO
    fibonacci := fibonacci(n - 1) + fibonacci(n - 2)
  FIN_SI
FIN_FUNCION
```
