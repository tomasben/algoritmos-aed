## 5.1.5
Dados dos números enteros, divídalos (división entera) y muestre el resultado,
usando sólo la operación resta.

$$
div(a, b) =
\begin{cases}
  0, & \text{si } a < b \\
  1, & \text{si } a = b \\
  1 + div(a - b, b), & \text{si } a > b
\end{cases}
$$

```
ACCION ejercicio ES
  AMBIENTE
    a, b, c: entero

    FUNCION div(a, b: entero): entero ES
      SI (a < b) ENTONCES
        div := 0
      CONTRARIO
        SI (a = b) ENTONCES
          div := 1
        CONTRARIO
          div := 1 + div(a - b, b)
        FIN_SI
      FIN_SI
    FIN_FUNCION
  PROCESO
    ESCRIBIR("Ingrese el dividendo: ")
    LEER(a)

    ESCRIBIR("Ingrese el divisor: ")
    LEER(b)

    c := div(a, b)
    ESCRIBIR("El número ", a, " puede ser dividido ", c, " veces por ", b)
FIN_ACCION
```
