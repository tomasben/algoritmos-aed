## 5.1.1
Calcular el factorial de un número positivo n. Tener en cuenta la definición
matemática de n!

$$
n! =
\begin{cases}
    1 & \text{si } x = 0 \\
    n * (n - 1) & \text{si } x > 0
\end{cases}
$$

```
FUNCION factorial(n: entero): entero ES
  SI (n = 0) ENTONCES
    factorial := 1
  CONTRARIO
    factorial := n * factorial(n - 1)
  FIN_SI
FIN_FUNCION
```
