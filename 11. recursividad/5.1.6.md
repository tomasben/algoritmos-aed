## 5.1.6
Determine recursivamente si un número dado es par o impar, usando sólo la
operación resta.

$$
par(n) =
\begin{cases}
  verdadero, & \text{si } n = 0 \\
  falso, & \text{si } n = 1 \\
  par(n - 2), & \text{si } n > 1
\end{cases}
$$

```
FUNCION par(n: entero): logico ES
  SI (n > 1) ENTONCES
    par := par(n - 2)
  CONTRARIO
    SI (n = 0) ENTONCES
      par := verdadero
    CONTRARIO
      par := falso
    FIN_SI
  FIN_SI
FIN_FUNCION
```
