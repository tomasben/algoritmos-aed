## 5.1.4
Construir un algoritmo recursivo que permita determinar si los dígitos de un
número n dado son todos pares.

$$
par(n) =
\begin{cases}
  \text{verdadero}, & \text{si } 0 \le n \le 9 \text{ y } (n \text{ MOD } 2) = 0 \\
  \text{falso}, & \text{si } 0 \le n \le 9 \text{ y } (n \text{ MOD } 2) <> 0 \\
  ((n \text{ MOD } 10) \text{ MOD } 2) = 0 \text{ y } par(n \text{ DIV } 10), & \text{si } n \ge 10
\end{cases}
$$

```
FUNCION par(n: entero): lógico ES
  SI (n en 0..9) ENTONCES
    SI ((n MOD 2) = 0) ENTONCES
      par := verdadero
    CONTRARIO
      par := falso
    FIN_SI
  CONTRARIO
    par := (((n MOD 10) MOD 2) = 0) Y par(n DIV 10)
  FIN_SI
FIN_FUNCION
```
