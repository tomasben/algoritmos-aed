## 5.2.1
Imprimir las cifras de un nÃºmero n (siendo n >= 0) en orden inverso al original.
Por ej.: el inverso de 254 es 452.

$$
\text{inverso}(n) =
\begin{cases}
  \text{ESCRIBIR}(n), & \text{si } n < 10 \\
  \text{ESCRIBIR}(n \text{ MOD } 10) \text{ y } \text{inverso}(n \text{ DIV } 10), & \text{si } n \ge 10
\end{cases}
$$

```
PROCEDIMIENTO inverso(n: entero) ES
  SI (n >= 10) ENTONCES
    ESCRIBIR(n MOD 10)
    inverso(n DIV 10)
  CONTRARIO
    ESCRIBIR(n)
  FIN_SI
FIN_PROCEDIMIENTO
```
