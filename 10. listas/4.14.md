## 4.14
Se desea ingresar una serie de números, e imprimirlos en el orden contrario al
de ingreso. Diseñar un algoritmo que satisfaga tal requerimiento, utilizando
la estructura de datos más apropiada.

```
ACCION ejercicio ES
  AMBIENTE
    Nodo = REGISTRO
      dato: entero
      prox: puntero a Nodo
    FIN_REGISTRO
    
    prim, p, q: puntero a Nodo
    num: entero
    continuar: caracter
  PROCESO
    prim := nil
    
    REPETIR 
      ESCRIBIR("Ingrese un número para añadirlo a la lista: ")
      LEER(num)
      
      NUEVO(q)
      *q.dato := num
      
      // carga apilada
      // prim vale nil en la 1er entrada
      *q.prox := prim
      prim := q
    
      ESCRIBIR("Desea ingresar otro número? [s/n]")
      LEER(continuar)
    HASTA QUE (continuar = 'n')
    
    p := prim
    MIENTRAS (p <> nil) HACER
      ESCRIBIR(*p.dato)
      p := *p.prox
    FIN_MIENTRAS
FIN_ACCION
```
