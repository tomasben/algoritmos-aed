## 3.22
Dada una matriz cuadrada de 5 x 5 de n√∫meros, sumar filas y columnas y guardar
en una matriz de 2 x 5, de modo que la fila 1 contenga la suma de cada fila y
la fila 2, la suma de cada columna.

```
ACCION ejercicio ES
  AMBIENTE
    matriz: arreglo [1..5, 1..5] de real
    totales: arreglo [1..2, 1..5] de real
    i, j: entero
    fila, columna: real
  PROCESO
    PARA j := 1 HASTA 5 HACER
      fila := 0
      PARA i := 1 HASTA 5 HACER
        fila := fila + matriz[j, i]
      FIN_PARA
      totales[1, j] := fila
    FIN_PARA

    PARA i := 1 HASTA 5 HACER
      columna := 0
      PARA j := 1 HASTA 5 HACER
        columna := columna + matriz[j, i]
      FIN_PARA
      totales[2, i] := columna
    FIN_PARA
FIN_ACCION
```
