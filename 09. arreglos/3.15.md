## 3.15
Se precisa ordenar un arreglo de enteros de menor a mayor, eliminando los nÃºmeros repetidos.

```
ACCION ejercicio ES
  AMBIENTE
    i, izq, der: entero
    arr: arreglo [1..n] de entero
    fue_modificado: logico
  PROCESO
    REPETIR
      fue_modificado := falso
      PARA i := 1 HASTA (n - 1) HACER
        izq := arr[i]
        der := arr[i + 1]

        SI izq > der ENTONCES
          arr[i] := der
          arr[i + 1] := izq
          fue_modificado := verdadero
        CONTRARIO
          SI izq = der ENTONCES
            arr[i + 1] := 0
          FIN_SI
        FIN_SI
      FIN_PARA
    HASTA QUE fue_modificado = falso
FIN_ACCION
```
